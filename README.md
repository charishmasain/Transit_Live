🚍 Smart College Transport Tracking System

✨ Overview

A smart web-based application that enables students and faculty to track college buses in real time, view estimated arrival times (ETA), and plan their commute efficiently.

🚀 Features

🗺️ Live bus tracking on interactive map (simulated)
⏱️ Real-time ETA updates
🔔 Smart alerts for bus arrival
🧑‍🎓👩‍🏫 Supports both students and faculty
📍 Clean and user-friendly interface
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Maps: Leaflet.js (OpenStreetMap)
Type: Web Application (Prototype)
🎯 Problem

Unpredictable bus timings cause:

⏳ Long waiting times
❌ Missed buses
😕 Lack of real-time information
💡 Solution

This system allows users to:

📍 Track bus location live
⏱️ Check ETA instantly
📊 Plan their commute better
